Zone

Manage DNS zones including creation, update, deletion, and retrieval.

List all DNS zones

query Parameters
name
string

Filter by zone name.

id
integer <int32>

Filter by zone ID.

primary_ns
string

Filter by primary name server.

admin_email
string

Filter by admin email.

ttl
integer <int32>

Filter by TTL.

min_ttl
integer <int32>

Filter by minimum TTL.

max_ttl
integer <int32>

Filter by maximum TTL.

serial
integer <int32>

Filter by serial.

search
string

Partially search zones.

limit
integer <int32>
